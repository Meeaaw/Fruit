Welcome to Fruit Taps

By: Sam, Mia, and Vincent

This is an interactive game where you tap on fruit to score points.

Documentation

This project is based on an old mobile game called Fruit Ninjas. Below is the process of how we developed our game

  • Prototype: We started off with a very simple sketch
  • Making of the background: We started polishing the game, making it more visually appealing.
  • Making of the fruits: Initally we were going to use real fruits, but we came across an issue that not all fruits are conductive, so we ended up using fake fruits.
